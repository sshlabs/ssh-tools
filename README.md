# ssh-tools

## Building

To build the container that then allows you to generate a ssh key in an isolated environment, use the following:

```
docker build -t sshlabs/ssh-server:1.0 .
```

## Running

To run the container created using the above command, use the following command:

```
docker run --rm sshlabs/ssh-server:1.0
```
